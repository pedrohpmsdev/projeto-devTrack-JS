# projeto-devTrack-JS

DevTrack é um gerenciador de projetos e tarefas para desenvolvedores, usado diretamente no terminal. Ele começa como um simples script com readline, evolui para um CLI profissional com Commander.js, ganha uma API REST local, integração com GitHub, estruturas de dados sofisticadas e, ao final, é documentado e preparado para publicação no npm.
